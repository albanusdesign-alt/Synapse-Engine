API Keys
Stored only in this browser on this device — never in the file's code. Keys leave your machine only in the direct calls to Google and Anthropic.
🗺️ Google Places API Key (required)
Get one: console.cloud.google.com → APIs & Services → Credentials → Create key. Enable Places API (New) and Maps JavaScript API, turn on billing, and allow your localhost origin.
✨ Anthropic API Key (required for research & pitch)
Get one: console.anthropic.com → API Keys → Create key. Deep Research uses web search — roughly a few cents per business.
🔍 Lusha API Key (optional)
Verified contact enrichment. Server-only API — browsers block direct calls (CORS), so this works only through a proxy.
🧭 Explorium API Key (optional)
Firmographic / prospect enrichment. Same as Lusha: server-only, needs a proxy from the browser.
Serve this over http://localhost (any static server) — Google Maps won't run from a file:// page.
